Board.py
Core module implementing Catan board mechanics.
Classes
Tile

Represents a single hex tile
Attributes: resource_type, value (2-12, None for desert)

Board

Manages game board state and layout
Generates and tracks 19 hexagonal tiles (3-4-5-4-3 pattern)
Handles tile adjacency and coordinate calculations

Resource Distribution

4x: WOOD, GRAIN, WOOL
3x: BRICK, ORE
1x: DESERT

Dependencies

.enums: ResourceType
.constants: TILE_SIZE, SCREEN_WIDTH, SCREEN_HEIGHT

Could be split into: tile.py, board_generator.py, board_geometry.py
